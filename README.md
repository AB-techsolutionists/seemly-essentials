# Seemly Essentials

Shopify theme customization project.

## Workflow

1. Pull merchant theme: `shopify theme pull --store <store>`
2. Customize local files: Liquid, CSS, JS, JSON
3. Preview: `shopify theme dev`
4. Push changes: `shopify theme push`
5. Publish: `shopify theme publish`

## Requirements

- Shopify CLI
- Collaborator access with Manage themes permission
